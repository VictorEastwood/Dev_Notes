## 一、 修饰键符号说明

| 符号 | 对应按键名称 | Windows 对应键 |
| --- | --- | --- |
| **⌘** | Command (Cmd) | Windows 徽标键 |
| **⌥** | Option (Alt) | Alt 键 |
| **⌃** | Control (Ctrl) | Ctrl 键 |
| **⇧** | Shift | Shift 键 |
| **⇪** | Caps Lock | 大写锁定键 |
| **🌐 / Fn** | Function / 地球仪键 | - |
| **⎋** | Esc | Esc |
| **⏎** | Return | Enter / 回车 |
| **⌫ / ⌦** | Delete / 向前删除 | Backspace / Delete |

## 二、 通用常用快捷键

* **⌘ X**: 剪切所选项
* **⌘ C**: 拷贝所选项
* **⌘ V**: 粘贴内容
* **⌘ Z**: 撤销（**⇧ ⌘ Z** 重做）
* **⌘ A**: 全选
* **⌘ F**: 查找项目
* **⌘ G**: 再次查找（**⇧ ⌘ G** 查找上一处）
* **⌘ H**: 隐藏当前 App 窗口（**⌥ ⌘ H** 隐藏其他 App）
* **⌘ M**: 最小化当前窗口（**⌥ ⌘ M** 最小化 App 所有窗口）
* **⌘ O**: 打开所选项
* **⌘ P**: 打印当前文稿
* **⌘ Q**: 退出当前 App
* **⌘ S**: 存储当前文稿
* **⌘ T**: 打开新标签页
* **⌘ W**: 关闭最前方窗口（**⌥ ⌘ W** 关闭 App 所有窗口）
* **⌥ ⌘ ⎋**: 强制退出 App
* **⌘ 空格键**: 显示/隐藏聚焦搜索
* **⌥ ⌘ 空格键**: 在访达中进行聚焦搜索
* **⌃ ⌘ 空格键** 或 **🌐 E**: 显示字符检视器（表情符号）
* **⌃ ⌘ F**: 全屏模式切换
* **空格键**: 快速查看预览
* **⌘ ⇥**: 切换到下一个最近使用的 App
* **⌘ `**: 在当前 App 的窗口间切换
* **⇧ ⌘ 5**: 截屏或录屏（**⇧ ⌘ 3/4** 亦可）
* **⌘ ,**: 打开当前 App 设置
* **🌐 Q**: 创建快速备忘录

## 三、 访达 (Finder) 与系统快捷键

### 文件管理

* **⌘ D**: 复制所选文件
* **⌘ E**: 推出所选磁盘
* **⌘ I**: 显示简介
* **⌘ R**: 显示替身原文件 / 刷新页面
* **⇧ ⌘ N**: 新建文件夹
* **⌃ ⌘ N**: 用所选项目新建文件夹
* **⌥ ⌘ V**: 移动文件（剪切粘贴）
* **⌘ ⌫**: 移到废纸篓
* **⇧ ⌘ ⌫**: 清倒废纸篓（**⌥ ⇧ ⌘ ⌫** 跳过确认）
* **⌘ Y**: 快速查看预览（**⌥ ⌘ Y** 幻灯片放映）

### 视图与跳转

* **⌘ 1/2/3/4**: 以图标/列表/分栏/画廊方式显示
* **⌘ [ / ]**: 前往上一个/下一个文件夹
* **⌘ ↑**: 打开父文件夹（**⌃ ⌘ ↑** 在新窗口打开）
* **⌘ ↓**: 打开所选项目
* **⇧ ⌘ C**: 打开“电脑”窗口
* **⇧ ⌘ D**: 打开“桌面”文件夹
* **⇧ ⌘ F**: 打开“最近使用”窗口
* **⇧ ⌘ G**: 打开“前往文件夹”
* **⇧ ⌘ H**: 打开个人文件夹
* **⇧ ⌘ I**: 打开 iCloud 云盘
* **⇧ ⌘ K**: 打开“网络”窗口
* **⌥ ⌘ L**: 打开“下载”文件夹
* **⇧ ⌘ O**: 打开“文稿”文件夹
* **⇧ ⌘ R**: 打开“隔空投送”窗口
* **⇧ ⌘ U**: 打开“实用工具”文件夹

### 界面显隐

* **⌥ ⌘ D**: 显示/隐藏程序坞 (Dock)
* **⇧ ⌘ P**: 显示/隐藏预览面板
* **⇧ ⌘ T**: 显示/隐藏标签页栏
* **⌥ ⌘ P**: 显示/隐藏路径栏
* **⌥ ⌘ S**: 显示/隐藏边栏
* **⌘ /**: 显示/隐藏状态栏
* **⌥ ⌘ T**: 显示/隐藏工具栏

### 系统交互

* **🌐 A**: 显示/隐藏程序坞
* **🌐 C**: 显示/隐藏控制中心
* **🌐 D**: 开始/停止听写
* **🌐 N**: 显示/隐藏通知中心
* **🌐 ⇧ A**: 显示隐藏 App (Tahoe 26+) 或启动台
* **⌘ + ⌘ (双击)**: 开启“通过键入使用 Siri”
* **⌥ 亮度键**: 打开显示器设置
* **⌥ 音量键**: 打开声音设置
* **⌥ 键盘亮度键**: 打开键盘设置

## 四、 文本编辑快捷键

* **⌘ B/I/U**: 粗体/斜体/下划线切换
* **⌘ K**: 添加网页链接
* **⌘ T**: 显示/隐藏字体窗口
* **⌃ ⌘ D**: 显示字词定义
* **⌥ ⌫**: 删除左侧词语
* **⌃ H**: 删除左侧字符（等同 **⌫**）
* **⌃ D**: 删除右侧字符（等同 **🌐 ⌫**）
* **⌃ K**: 剪切至行尾（**⌃ Y** 粘贴）
* **🌐 ↑ / ↓**: 上翻/下翻页 (Page Up/Down)
* **🌐 ← / →**: 移至开头/结尾 (Home/End)
* **⌘ ↑ / ↓**: 移至文稿开头/末尾
* **⌘ ← / →**: 移至行首/行尾
* **⌥ ← / →**: 移至上一词首/下一词尾
* **⇧ ⌘ ↑ / ↓ / ← / →**: 选中至文稿或行首尾
* **⌥ ⇧ ↑ / ↓ / ← / →**: 选中至段落或词语首尾
* **⌃ A / E**: 移至行/段落开头或末尾
* **⌘ { / }**: 左/右对齐
* **⇧ ⌘ |**: 居中对齐
* **⌥ ⌘ C / V**: 拷贝/粘贴样式
* **⌥ ⇧ ⌘ V**: 粘贴并匹配周围样式
* **⇧ ⌘ - / +**: 缩小/放大所选项目

## 五、 睡眠、退出登录与关机

* **电源按钮**: 点按开机/唤醒；按住 1.5 秒睡眠；长按强制关机
* **触控 ID**: 锁定屏幕
* **⌃ ⇧ 电源键**: 显示器睡眠
* **⌃ 电源键**: 显示重启/睡眠/关机对话框
* **⌃ ⌘ 电源键**: 强制重启（不提示存储）
* **⌃ ⌥ ⌘ 电源键**: 退出所有 App 并关机
* **⌃ ⌘ Q**: 立即锁定屏幕
* **⇧ ⌘ Q**: 退出登录用户账户（**⌥ ⇧ ⌘ Q** 立即退出）

## 六、 辅助功能快捷键

* **⌥ ⌘ F5** 或 **连按三下触控 ID**: 显示辅助功能快捷键面板
* **⌃ ⌥ ⌘ 8**: 反转颜色
* **⌃ ⌥ ⌘ , / .**: 降低/增强对比度
* **⌃ F2**: 焦点移至菜单栏
* **⌃ F3**: 焦点移至程序坞
* **⌃ F4**: 焦点移至活跃窗口
* **⌃ F5**: 焦点移至窗口工具栏
* **⌃ F8**: 焦点移至状态菜单
* **⇥ / ⇧ ⇥**: 移至下一个/上一个控制选项
